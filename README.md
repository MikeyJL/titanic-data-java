# Titanic Data (Java)

Converted a university project from cpp to java. Find the [**original here**](https://github.com/MikeyJL/university-com728/tree/main/data/titanic).

## Usage

### Compile

```bash
javac **/*.java
```
### Execute

```bash
java Main
```
